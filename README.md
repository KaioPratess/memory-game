## Overview

![image](https://user-images.githubusercontent.com/91703674/192334449-13563f81-baf7-4361-ade2-78d7b07ed2d8.png)


This project is a simple memory game created using the Marvel API. Don't click at the same character twice!

### Built With

- [ReactJs](https://reactjs.org/)
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)
- [Vite](https://vitejs.dev/)

## Features

This application/site is being created as a submission to a [The Odin Project](https://www.theodinproject.com) challenge. The [challenge](https://www.theodinproject.com/lessons/node-path-javascript-memory-card) was to build a memory game.

- ScoreBoard (current score, best score)
- The Best Score is saved in the local storage
- The photos are taken from the marvel API via fetch

## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com), [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) and [GitHub CLI](https://cli.github.com/) installed on your computer. From your command line:

```bash
#Log into Github
$ gh auth login --web

#Fork this repository and clone your fork
$ gh repo fork https://github.com/KaioPratess/memory-game.git --clone

# Install dependencies
$ npm install

# Run the app
$ npm start dev
```
